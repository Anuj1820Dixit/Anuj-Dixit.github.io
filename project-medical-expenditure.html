Project Summary

This project develops an automated, explainable system to estimate annual medical expenditure for new customers of ACME Insurance Inc., using demographic and lifestyle data such as: Age, Sex, BMI, Number of Children, Smoking Habit, Region of Residence. These predictions are used to determine the customer's insurance premium, and comply with regulatory requirements for transparency and interpretability.

  • Analysis of smoking habits, regional differences, and gender-based comparisons
  • Correlation matrix to identify key features affecting medical charges
üß† Modeling
  • Preprocessing: encoding categorical variables, creating interaction terms (e.g., age * BMI)
  • Models: Linear Regression (interpretability), Random Forest Regressor (accuracy), Gradient Boosting Regressor (performance)
  • Standardized feature scaling for model convergence
üöÄ Deployment
  • Django web application with a two-step user interface
  • Step 1: Calculate BMI by entering height and weight
  • Step 2: Enter region, age, children, smoking status
  • Prediction: System predicts annual medical charges and displays the result
  • Interactive, user-friendly onboarding for ACME Insurance customers
